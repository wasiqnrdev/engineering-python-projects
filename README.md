# Engineering Python Projects

A collection of Python projects exploring mechanics, aerospace engineering, orbital dynamics, and numerical simulations.

## Projects

### 🚀 Projectile Motion Simulator

Simulates two-dimensional projectile motion under constant gravitational acceleration.

**Features**
- Supports Earth, Mars, Jupiter, and the Moon
- Calculates:
  - Time of Flight
  - Maximum Height
  - Horizontal Range
- Plots the projectile trajectory using NumPy and Matplotlib

---

### 🛰️ Orbital Mechanics Calculator

Calculates key orbital parameters and visualizes circular satellite orbits around different celestial bodies.

**Features**
- Supports Earth, Mars, Jupiter, and the Moon
- Calculates:
  - Orbital Velocity
  - Orbital Period
  - Escape Velocity
- Visualizes the selected planet and its circular orbit
- Uses Newtonian gravitational equations
