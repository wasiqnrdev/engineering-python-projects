# Engineering Python Projects

A collection of Python projects exploring mechanics, aerospace engineering, orbital dynamics, and numerical simulations.

## Projects

### 🚀 Projectile Motion Simulator

- Simulates projectile motion on Earth, Mars, Jupiter, and the Moon.
- Calculates:
  - Time of flight
  - Maximum height
  - Horizontal range
- Plots the projectile trajectory using NumPy and Matplotlib.
