# Orbital Mechanics Calculator

A Python application that performs basic orbital mechanics calculations and visualizes circular orbits around different celestial bodies.

## Features

- Select a celestial body:
  - Earth
  - Mars
  - Jupiter
  - Moon
- Enter the orbital altitude above the planet's surface.
- Calculates:
  - Orbital Velocity
  - Orbital Period
  - Escape Velocity
- Generates a visualization showing:
  - The selected planet
  - The circular orbital path
- Displays the calculated orbital parameters in the terminal.

## Physics

The simulator assumes:

- Circular orbits
- Point-mass gravity
- No atmospheric drag
- Newtonian mechanics

## Technologies Used

- Python
- NumPy
- Matplotlib

The program calculates:
