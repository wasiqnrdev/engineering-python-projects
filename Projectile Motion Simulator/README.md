# Projectile Motion Simulator

A Python application that simulates two-dimensional projectile motion under constant gravitational acceleration.

## Features

- Select between Earth, Mars, Jupiter, and Moon
- Calculates:
  - Time of Flight
  - Maximum Height
  - Horizontal Range
- Plots the projectile trajectory using Matplotlib

## Technologies

- Python
- NumPy
- Matplotlib

## Physics Assumptions

- No air resistance
- Constant gravitational acceleration
- Two-dimensional projectile motion

## Run

Run plot.py for simulation

Choose a planet, enter the initial velocity and launch angle, and the program will display the projectile's trajectory.
