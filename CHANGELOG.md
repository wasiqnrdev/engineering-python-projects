# Changelog

## Project 1 - Projectile Motion Simulator
- Simulates projectile motion on Earth, Mars, Jupiter and the Moon.
- Calculates time of flight, maximum height and horizontal range.
- Visualizes the trajectory using Matplotlib.

## Project 2 - Orbital Mechanics Calculator
- Simulates circular orbits around the Earth, Moon, Jupiter and Mars
- Calculates:
  - Orbital Velocity
  - Orbital Period
  - Escape Velocity
- Visualizes the selected planet and the circular orbital path of the object using matplotlib
